Cyber Dictionary

A simple guide to common cybersecurity terms.

Phishing

A type of social engineering attack where a cybercriminal pretends to be a trusted entity to trick individuals into revealing sensitive information, like passwords or credit card numbers, usually through email.

Malware

Short for malicious software, this is any program or file designed to harm or exploit a computer, network, or data. Examples include viruses, worms, and trojans.

Firewall

A network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules. It acts as a barrier between your internal network and the public internet.

Ransomware

A type of malicious software that blocks access to a user's computer files until a ransom is paid. It often spreads through phishing emails or malicious websites.

Encryption

The process of converting information into a code to prevent unauthorized access. Only those with the correct key can decode and read the information.

MFA (Multi-Factor Authentication)

A security system that requires two or more verification methods to grant access to an account, such as a password and a code sent to your phone. This makes it much harder for hackers to access your account.
